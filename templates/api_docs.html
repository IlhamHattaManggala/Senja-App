Dokumentasi API Senja

Authentication
POST /register
Registrasi user baru. Field: name, email, password
✅ Response 201: 
{
  "status": "sukses",
  "pesan": "Registrasi berhasil",
  "data": {
    "user": {
      "id": "123",
      "name": "Nama",
      "email": "user@example.com"
    }
  }
}
⚠️ Response 400: 
{
  "pesan": "Semua field wajib diisi"
}
⚠️ Response 409: 
{
  "pesan": "Email sudah digunakan"
}
POST /login
Login user dan dapatkan token JWT. Field: email, password
✅ Response 200: 
{
  "status": "sukses",
  "pesan": "Login berhasil",
  "data": {
    "user": {
      "id": "123",
      "name": "Nama",
      "email": "user@example.com",
      "role_id": 2
    },
    "token": "jwt-token"
  }
}
⚠️ Response 401: 
{
  "pesan": "Email atau password salah"
}
POST /logout
Logout user dari aplikasi (tidak menghapus token).
✅ Response 200: 
{
  "pesan": "Logout berhasil"
}
Profile
GET /profile
Mengambil data profil user. Header butuh: Authorization: Bearer <token>
✅ Response 200: 
{
  "status": "sukses",
  "data": {
    "id": "123",
    "name": "Nama",
    "email": "user@example.com",
    "role_id": 2,
    "avatar": null
  }
}
⚠️ Response 401 (token tidak ditemukan): 
{
  "pesan": "Token tidak ditemukan"
}
⚠️ Response 401 (token expired): 
{
  "pesan": "Token expired"
}
⚠️ Response 401 (token invalid): 
{
  "pesan": "Token invalid"
}
⚠️ Response 404: 
{
  "pesan": "User tidak ditemukan"
}
Kategori
POST /category
Menambahkan kategori baru ke dalam database. Field: name_category
✅ Response 201: 
{
    "status": "sukses",
    "pesan": "Kategori berhasil ditambahkan",
    "data": {
      "id": "67f871b68325e0c2c218f1bd",
      "name_category": "Mudah"
    }
  }
⚠️ Response 400: 
{
    "pesan": "Kategori tidak boleh kosong"
  }
⚠️ Response 500: 
{
    "pesan": "Terjadi kesalahan pada server"
  }
